Tipos de selectores que entran en examen

  1. Clase (class=””)

    2. Una clase es una definición de un estilo que en principio no está asociado a alguna etiqueta HTML, pero que podemos asociar a etiquetas concretas.
    Mediante la definición de "clases'"" se establecen estilos que pueden aplicarse a cualquier etiqueta de HTML.
    El estilo no se vincula a una etiqueta concreto sino a una "clase",esta clase, actua como atributo de cualquier etiqueta, cada vez que se quiera llamar
    a ese estilo, basta con incluir el nombre de esa clase dentro de la etiqueta correspondiente.

    Ejemplo

    La palabra ejemplo la he puesto azul con class="" poniendo entre las dos etiquetas de style un .nombre.

  2. Identificador (id=””)

    3. En un documento HTML, los selectores de ID de CSS buscan un elemento basado en el contenido del atributo id. El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.

    Ejemplo

    La palabra ejemplo la he puesto roja con id poniendo entre las dos etiquetas style un #Identificador.

  3. Todos los elementos (*)

    4. Se llama selector universal. Se utiliza para seleccionar todos los elementos de la página.

    Ejemplo

    Todo el texto de esta página ahora está en arial.

  4. Varios elementos ( p, em, h1, h2)

    5. Se llama selector de etiqueta. Selecciona todos los elementos de la página cuya etiqueta HTML coincide con el valor del selector.

    Ejemplo

    Todos los elementos con etiqueta li ahora están en cursiva.

  5. Elementos contenidos en otros elementos (ol li)

    6. Todos los elementos dentro de las etiquetas p que a su vez están dentro de una etiqueta strong están en morado.

  6. Sobrevuleo (p:hover)

    7. La pseudo-clase: hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor.

    Ejemplo

    Todo el texto pertenecientes a la etiqueta p está en color verde cuando pasas el cursor por encima del texto correspondiente.
